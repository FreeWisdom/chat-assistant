# Python 基础 — 变量与数据类型

## 变量
Python 中变量不需要声明类型，直接赋值即可：
```python
name = "张三"    # 字符串
age = 20         # 整数
height = 1.75    # 浮点数
is_student = True # 布尔值
```

## 基本数据类型
- **int** 整数：如 1, -5, 1000
- **float** 浮点数：如 3.14, -0.5
- **str** 字符串：如 "hello", 'world'
- **bool** 布尔值：True 或 False
- **list** 列表：如 [1, 2, 3]
- **tuple** 元组：如 (1, 2, 3)
- **dict** 字典：如 {"name": "张三", "age": 20}
- **set** 集合：如 {1, 2, 3}

## 类型转换
```python
# 字符串转整数
num_str = "42"
num = int(num_str)

# 整数转字符串
str_num = str(42)

# 浮点数转整数（截断）
int_val = int(3.9)  # 结果为 3
```

## 常见问题
1. **NameError** — 变量未定义就使用，确保先赋值
2. **TypeError** — 类型不匹配，如字符串和数字直接相加
3. 使用 `type()` 函数可以查看变量类型
